# Grocery Management System

## Project Overview

The Grocery Management System is a full-stack web application developed to manage grocery products efficiently through a modern dashboard interface. The application allows users to add, edit, delete and manage grocery items while demonstrating full-stack development concepts including frontend development, backend API integration, authentication and database operations.

This project was developed as part of my full-stack learning journey to strengthen my practical skills in React, TypeScript, API development and modern web application architecture.

---

# Features

- User authentication and protected routes
- Add new grocery products
- Edit existing products
- Delete products
- Product listing dashboard
- Responsive user interface
- REST API integration using Axios
- Backend middleware authentication
- Component-based frontend architecture
- Full CRUD functionality
